# JI23-DEV

英語の品詞を覚えるためのクイズアプリです。

## 機能

主な機能は以下になります。

- クイズ
- 英単語の音声再生

## 構成

- React / Next.js / TypeScript / Vercel / microCMS で JAMstack な CMS 構築
- UI ライブラリは Chakra UI

## ビルド環境

- [Node.JS](https://nodejs.org) >= 14.17.5
- [Yarn](https://yarnpkg.com) >= 1.16.0

## スクリプト

`yarn` で実行可能なスクリプトの一覧です。

### dev

開発サーバーを立ち上げます。

### build

本番環境用のファイルをビルドします。
